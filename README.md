🎓 **UniSwipe** is a Tinder-style university discovery platform that helps students explore and compare universities in a more interactive way.

The idea behind the project was to simplify the university search process by allowing users to swipe through universities, save the ones they like, apply filters, and compare options using a recommendation engine.

The frontend was built using Next.js, React, and TypeScript with Framer Motion animations for the swipe-card experience, while the backend was developed using FastAPI and Python.

The platform includes:

* Swipe-based university browsing
* Saved universities system using localStorage
* Search and filtering by country, program, and co-op availability
* University comparison system
* Intelligent recommendation engine that suggests:

  * Best Overall
  * Best for Budget
  * Best for Career/Co-op
  * Best Global Ranking

### How to Run / Test the Project

#### Frontend

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

#### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Open API docs:

```text
http://127.0.0.1:8000/docs
```

To test the project:

1. Start both frontend and backend
2. Open the frontend in the browser
3. Swipe through universities
4. Save a few universities
5. Apply filters/search
6. Compare selected universities
7. View recommendation results generated by the scoring engine
